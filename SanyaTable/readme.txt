Vendor Project Plugin
=====================

Описание
--------

Стартовый плагин работающий Ignet Dev Toolkit.
Перед использованием развернуть composer-зависимости в Ignet Dev Toolkit.

Как переписать readme при релизе плагина
----------------------------------------

Развернуто опишите здесь функционал плагина, используемые компоненты и библиотеки,
желательно с указанием версий и ссылок на документацию.

Все хитрое, сложное, важное, интересное стоит здесь описать.
Этот файл пишется для разработчиков или клиентов способных делать что-либо с кодом.
